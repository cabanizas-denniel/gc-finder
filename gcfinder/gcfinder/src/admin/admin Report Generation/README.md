# Excel Report Generator

This Python script generates a sample sales report in Excel format with formatted data.

## Requirements

- Python 3.x
- Required packages: pandas, openpyxl

## Installation

Install the required packages using:

```bash
pip install -r requirements.txt
```

## Usage

Simply run the script:

```bash
python generate_report.py
```

This will generate a `sales_report.xlsx` file in the current directory with sample sales data.
